# ☕ Coffee Making Interactive Project

This project is an interactive infographic website that explains the **coffee making process** from farming to brewing.  
The project focuses on scroll animation, UI transitions, and infographic storytelling using HTML, CSS, JavaScript, and Anime.js.

---

## 📌 Project Overview

The goal of this project is to create a visually engaging experience where users can learn how coffee is made step-by-step through animation and interactive elements.

The website includes:

- Start screen with transition animation
- Multiple slides showing coffee production stages
- Scroll-based animation
- Number counter animation
- Image component separation for animation
- Responsive layout
- 3D model

---

## ☕ Coffee Making Process Included

1. Planting coffee seedlings  
2. Harvesting coffee cherries  
3. Processing beans  
4. Drying beans  
5. Roasting coffee  
6. Grinding coffee  
7. Brewing coffee

Each step is displayed as an animated infographic slide.
